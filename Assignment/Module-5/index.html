  • Q1.What is jquery?

    Ans. Jquery is a JavaScript liabrary. the slogan of jquery is "write less do more". It's main object to simplyfy javaScript complex structure.

      Features:
    • HTML/DOM Manipulation: jQuery allows you to manipulate HTML elements and their properties easily.
    • CSS Manipulation: You can modify CSS styles dynamically using jQuery.
    • HTML Event Methods: jQuery provides methods to handle HTML events (e.g., clicks, mouse movements).
    • Effects and Animations: You can create smooth animations and transitions.
    • AJAX: jQuery simplifies making asynchronous requests to the server (e.g., fetching data without reloading the page).

  • Q2. How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery ,JQuery Animation?

    Ans.

    • To set a specific CSS property for an element, use the css() method.Example
      $('selectorname').css("property name","value");
    • Adding CSS Classes Using jQuery:
      To add CSS classes to an element, you can use the addClass() method. It allows you to add one or more classes to selected elements.
      $("selector").addClass("class_name");
    • Removing CSS Classes Using jQuery:
      To remove specific CSS classes from elements, use the removeClass() method
      $("selector").removeClass("class_name");
    • Animating Elements Using jQuery:
      While not directly related to CSS classes, jQuery also provides animation methods.
      You can animate properties like width, height, opacity, etc.
      $("button").click(function() {
      $("h1").animate({ fontSize: "24px" }, 1000);
      });
